# Phase 4 図生成：各図と使用セクション対応表

## 概要

Phase 4 では、Phase 3 成果物（Markdown ドキュメント）から対象セクションをルールベースで抽出し、
各セクションを LLM に渡して PlantUML 図を生成する。
本資料はどの図がどのセクション情報をもとに生成されるかをまとめたものである。

---

## 各図と使用セクションの対応

| # | 生成する図 | 図種別 | 使用プロンプト | 主セクション | 補完セクション | 挿入位置 |
|---|-----------|--------|--------------|------------|--------------|---------|
| 1 | 内部構成図 | クラス図 | `phase4_class_diagram.md` | `### 2.1 内部構成` | `## 3. インタフェース` | 2.1セクション末尾 |
| 2 | 外部依存関係図 | コンポーネント図 | `phase4_component_diagram.md` | `### 2.2 外部依存関係` | なし | 2.2セクション末尾 |
| 3 | データ構造図 | クラス図 | `phase4_class_diagram.md` | `## 4. データ定義`（章全体） | なし | 4章末尾 |
| 4 | 全体処理フロー図 | アクティビティ図 | `phase4_activity_diagram.md` | 5章冒頭テキスト *1 | なし | 5章冒頭テキスト末尾 |
| 5 | サブセクション処理フロー図 | アクティビティ図 / 状態遷移図 / シーケンス図 *2 | `phase4_subsection_diagram.md` | 各 `### 5.X` サブセクション | なし | 各 5.X 末尾 |

*1: 小機能ありの場合は最初の `### 5.X` より前の冒頭テキスト。小機能なしの場合は5章全体。
*2: セクションの記述内容に応じて LLM が図種別を選択する。

---

## セクション抽出ルール詳細

### 1. 内部構成図（クラス図）

```
主: ### 2.1 内部構成 の先頭行から次の ### または ## の直前まで
補: ## 3. インタフェース の先頭行から次の ## の直前まで（章全体）
```

**補完情報の使い方**:
- 3章インタフェース情報は、2.1で定義されたクラス・モジュール間の関係（依存・呼び出し・データの受け渡し）を読み取るためにのみ使用する
- 3章に登場する要素を新たにクラス図に追加してはならない

**スキップ条件**:
- 2.1セクションが「該当なし」のみの場合は図を生成しない（3章の情報があっても生成しない）

### 2. 外部依存関係図（コンポーネント図）

```
### 2.2 外部依存関係 の先頭行から次の ### または ## の直前まで
```

### 3. データ構造図（クラス図）

```
## 4. データ定義 の先頭行から次の ## の直前まで（章全体）
```

小機能サブセクション（`### 4.X`）の有無に関わらず、4章全体を1つのクラス図として生成する。

### 4. 全体処理フロー図（アクティビティ図）

**小機能ありの場合**:
```
## 5. 処理フロー の先頭行から最初の ### 5.X の直前まで（冒頭テキストのみ）
```

**小機能なしの場合**:
```
## 5. 処理フロー の先頭行から次の ## の直前まで（章全体）
```

### 5. サブセクション処理フロー図

```
各 ### 5.X の先頭行から次の ### または ## の直前まで
```

小機能ありの場合のみ生成される。図種別（アクティビティ図・状態遷移図・シーケンス図）は
セクションの記述内容に応じて LLM が判断する。

---

## 生成スキップの条件

| 条件 | スキップの仕組み |
|------|----------------|
| 2.1セクションが「該当なし」のみ | セクション抽出時にルールベースで除外（LLMを呼ばない） |
| LLM が図を生成しないと判断 | LLM が空文字を返す |
| LLM が `@startuml`/`@enduml` のみ・`title` のみを返す | `_has_diagram_content()` で除外 |

## Phase 3 ドキュメント構造と対象セクションの位置

```
## 1. 機能概要                     ← 対象外
## 2. 機能構成
  ### 2.1 内部構成                 ← [図1] 主セクション
  ### 2.2 外部依存関係             ← [図2]
## 3. インタフェース               ← [図1] 補完セクション
  ### 3.X（小機能ありの場合）
## 4. データ定義                   ← [図3] 章全体
  ### 4.X（小機能ありの場合）
## 5. 処理フロー
  (冒頭テキスト)                   ← [図4] 小機能ありの場合
  ### 5.1 <小機能名>               ← [図5]
  ### 5.2 <小機能名>               ← [図5]
  ...
  (小機能なし: 章全体)             ← [図4]
## 6. 前提条件および制約           ← 対象外
## 7. 参考情報                     ← 対象外
## 8. 変更履歴                     ← 対象外
## 付録A〜C                        ← 対象外
```
